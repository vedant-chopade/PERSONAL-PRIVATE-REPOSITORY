Switching Between Multiple GitHub Accounts in Git Bash

Step 1: Remove All Loaded SSH Keys

eval "$(ssh-agent -s)"
        ssh-add -D

Step 2: Add the SSH Key for the Desired GitHub Account

For Primary GitHub Account

ssh-add ~/.ssh/id_ed25519

For Secondary GitHub Account

ssh-add ~/.ssh/id_ed25519_secondary

Step 3: Verify the Loaded SSH Key

ssh-add -l

Step 4: Test SSH Connection to GitHub

For Primary GitHub Account

ssh -T git@github.com

For Secondary GitHub Account

ssh -T git@github-second

Step 5: Configure Git User for Each Account

Set Git User for Primary Account

git config --global user.name "Primary Username"
git config --global user.email "primary-email@example.com"

Set Git User for Secondary Account

git config --global user.name "Secondary Username"
git config --global user.email "secondary-email@example.com"

Step 6: Clone or Push Repositories with the Correct Account

Clone a repo using the primary account

git clone git@github.com:primary-username/repository-name.git

Clone a repo using the secondary account

git clone git@github-second:secondary-username/repository-name.git

Switching Between Accounts

Whenever you need to switch accounts, repeat these steps from Step 1.
